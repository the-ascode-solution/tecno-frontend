# Placeholder Image URLs

## 300x200 Placeholder
https://via.placeholder.com/300x200/007bff/ffffff?text=300x200

## Usage Example
```jsx
<img src="https://via.placeholder.com/300x200/007bff/ffffff?text=300x200" alt="Placeholder" />
```

## Custom Placeholder Generator
- URL: https://via.placeholder.com/
- Format: https://via.placeholder.com/{width}x{height}/{background-color}/{text-color}?text={text}
- Example: https://via.placeholder.com/400x300/ff0000/ffffff?text=Sample+Image
